## Cruise Control Simulator

This Dash app visualizes a simple PID-based cruise control loop. Tune the controller gains, payload, and road conditions to see how the vehicle speed responds over time. TODO: fuzzy logic to add on second graph using simpful module.

### Requirements
- Python 3.10+
- `dash`, `dash-bootstrap-components`, `plotly`, `numpy`, `scipy`, `simpful`

Install dependencies and launch the app:

``` bash
pip install -r requirements.txt
python3 main.py
```

Open the provided local URL to interact with the dashboard.
